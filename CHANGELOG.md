## Version History

### Version numbering
- Major: Large feature additions, architecture changes
- Minor: New features, enhancements
- Patch: Bug fixes, documentation updates

**Examples:**
- 1.0.0: First iOS release on App Store
- 1.1.0: Add weather integration, news headlines
- 1.1.1: Fix voice synthesis crash on iOS 15
- 2.0.0: Add macOS support
