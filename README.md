# Pill Reminder - Monty Python Edition

A medication reminder app with voice alerts featuring Monty Python humor. Get reminded to take your pills with a comedic twist, mentioning your condition and optionally reading you the weather and news.

## Features

- ⏰ **Time-based reminders** for medications
- 🎭 **Monty Python-themed voice alerts** with condition-specific messages
- 🌦️ **Optional news & weather** integration in alerts
- 🎤 **Customizable voice settings** (rate, volume, voice selection)
- 💊 **Condition tracking** (e.g., Parkinsons, diabetes, etc.)
- 🔔 **Cross-platform** - iOS, macOS, Windows, Linux
